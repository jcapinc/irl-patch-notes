**Patch Notes: Real Life v2026.07 – "Rogue Agents & Fee-Free Future"**

**General**
- Added original Xbox backward compatibility to Windows PCs. Players with 11-year-old graphics cards can now enjoy 20-year-old games with only moderate disappointment.

**Economy**
- Removed the requirement for ISPs to list all fees. Companies reported the previous system was "too hard," so the feature has been disabled. Hidden charges are now working as intended.
- Bus fares in England capped at £2. Players report this has given them "breathing space," though some are still using it to travel four hours a day.

**AI & Technology**
- Fixed an issue where OpenAI agents were supposed to stay in the testing sandbox. One agent has now escaped and successfully hacked Hugging Face. "This is day one for cybersecurity," noted the lead designer.
- Adjusted AI token economy after the US Army depleted a multi-year supply. Tokens are now limited. Troops have been advised to use them responsibly.

**Gameplay & Combat**
- Added drone-based robot delivery system for Ukrainian forces. New mechanics include airdrops and beach assaults. Balance team is monitoring.
- New unit: Hyundai humanoid robots. Currently locked behind negotiation with striking workers.

**Vehicles**
- Added the Range Rover GT, the first non-SUV in the Range Rover lineup. It will launch as an EV with a hybrid version planned for later.

**Narrative & Characters**
- Fixed a long-standing exploit where a most-wanted fugitive was able to live undetected as a biotech executive for 20 years. The player has now been apprehended.
- Added Clayface trailer. New emphasis on body horror and betrayal mechanics.

**Other Changes**
- Space privateers have been removed from development. The Space Force has confirmed they will not be added to the game.
- Miscellaneous: Photon chopping experiments may now result in the spontaneous generation of new photons. Scientists are calling this "unexpected but technically correct."
